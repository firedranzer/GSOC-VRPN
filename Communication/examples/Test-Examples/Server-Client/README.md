# Server-Client

A Simple Server Client Communication via text Message.

## Running the example

* Make Sure that VRPN library is compiled correctly on your Machine.

### Starting the Server

```
$ g++ testClient.cpp
$ ./a.out
```

### Starting the Client

```
$ g++ testServer.cpp
$ ./a.out
```

* Note : If you face errors while compiling you can use flags like `-lvrpn` and `-lpthread`.
